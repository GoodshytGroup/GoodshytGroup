# GitHub Pages Deployment — Anime Battle Docs Hub

## Why

The interactive template uses CSS animations and JavaScript, which GitHub strips from rendered `README.md`.
GitHub Pages serves the raw HTML so the animation actually runs in a browser.

## How

1. Open the repo → **Settings** → **Pages**.
2. Under **Build and deployment**, choose **Deploy from a branch**.
3. Select **main** and **/ (root)**, then save.
4. Wait for the first deployment (usually under a minute).
5. Open:

```
https://goodshytgroup.github.io/Better-README/index-page-starters/templates/anime-battle/
```
